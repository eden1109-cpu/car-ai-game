# car-ai-game

スマホ縦画面向けの 5 車線 2D カー回避ゲームです。Canvas と Vanilla JS で動作します。

## 変更内容（今回）
- 敵車を3種類（sedan/truck/sport）追加し、速度とサイズを差別化。
- コイン出現・取得システムを追加。
- `localStorage` でスコア / ベストスコア / コイン / 所持スキン / 選択中スキンを保存。
- スキン購入・選択 UI を追加（開始パネル内）。
- 画像ファイルを使わず Canvas 上でドット絵風の車を描画。
- スマホ向けUIを改善（情報パネル、ボタンサイズ、safe-area対応）。
- 衝突・コイン取得・スピードアップ時のパーティクル / ラベル演出を追加。
- 将来機能拡張しやすいよう、`index.html` + `styles.css` + `game.js` に分割。

## 次にやるべきこと
1. **ランキング基盤**
   - スコア送信 API クライアント層を `services/leaderboard.js` として分離。
   - オフライン時のリトライキューを導入。
2. **ガチャ基盤**
   - スキン定義を JSON 化し、レア度・排出率を持たせる。
   - `coins` とは別のチケット通貨の導入。
3. **ゲームバランス調整**
   - 敵車種類ごとに出現テーブルを段階化。
   - コインの出現頻度をプレイ時間に応じて可変化。
4. **演出強化**
   - 擬似カメラシェイク、ヒットストップ、SEの追加。
5. **保守性向上**
   - `GameState`, `Renderer`, `Spawner`, `Storage` をクラス分割。
   - 最低限の E2E（Playwright）とユニットテストを導入。
